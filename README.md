# Unilink-APAYA-Kelompok-II
Kirana Isna Dewi (24082010147)-Pararel IV-D

UniLink adalah platform kemitraan berbasis Android yang dirancang khusus untuk memfasilitasi program magang antara mahasiswa dan sektor industri secara lebih transparan dan terintegrasi. Aplikasi ini menyederhanakan proses pencarian mitra perusahaan, memungkinkan mahasiswa untuk mengeksplorasi berbagai peluang karier profesional, serta membangun koneksi langsung dengan penyedia program magang dalam satu ekosistem digital yang efisien.

Daftar Fitur dan Deskripsi Fitur
1. Registrasi Pengguna
Fitur ini pengguna (mahasiswa maupun perusahaan) akan membuat akun baru di UniLink. Pengguna diminta mengisi data seperti nama, email, password.
2. Login Pengguna
Fitur login digunakan untuk mengakses akun yang telah terdaftar. Pengguna memasukkan email dan password yang valid untuk masuk ke dalam sistem.
3. Pencarian Lowongan Magang
Mahasiswa dapat mencari peluang magang berdasarkan kategori atau kata kunci tertentu. Fitur ini membantu menemukan lowongan yang sesuai dengan minat dan keahlian.
4. Profil Mahasiswa
Mahasiswa dapat membuat dan mengelola profil pribadi yang berisi data diri dan daftar koneksi.
5. Sistem Notifikasi
Menyediakan notifikasi untuk memberi informasi terkait pesan dari perusahaan, serta update penting lainnya agar pengguna tidak ketinggalan informasi.
6. Events
Fitur ini menampilkan daftar event dan workshop yang akan datang atau event yang dijadwalkan dalam waktu mendatang. Pengguna dapat melihat detail event seperti nama, deskripsi, tanggal, lokasi, dan penyelenggara.
7. Add Post (Berbagi Pengalaman)
Fitur ini memungkinkan pengguna untuk membuat postingan terkait pengalaman magang, tips karier, atau cerita inspiratif. Postingan dapat dilihat oleh pengguna lain sehingga membangun komunitas berbagi informasi dan pengalaman.
8. Connect (Jaringan Relasi)
Fitur ini menampilkan daftar orang yang mungkin dikenal oleh pengguna, seperti sesama mahasiswa, alumni, atau profesional dari perusahaan. Pengguna dapat memperluas jaringan dengan menambahkan koneksi baru.
9. Notifikasi Detail
Saat pengguna menekan notifikasi, akan ditampilkan detail notifikasi yang berisi informasi lengkap terkait aktivitas tertentu.
Tersedia opsi: Tandai Dibaca: untuk menghapus status notifikasi baru dan Tutup: untuk keluar dari detail notifikasi
